# ZeroInput

ZeroInput is a compact game booster profile kit for cleaner launches, calmer background load, and repeatable low-latency gaming sessions. It is designed as a transparent preset layer: every rule is visible, every profile is editable, and every run can be previewed before you apply anything.

## Highlights

| Area | What ZeroInput Helps With |
| --- | --- |
| Launch prep | Builds a pre-game checklist from the selected profile |
| FPS consistency | Surfaces settings that reduce avoidable background pressure |
| Input latency | Keeps focus on mouse, keyboard, overlay, and capture hygiene |
| Repeatability | Saves a small session report so every run can be compared |
| Safety | Runs in preview mode by default and does not edit the registry |

## Profiles

ZeroInput ships with three practical presets:

| Profile | Best For | Focus |
| --- | --- | --- |
| `balanced` | Daily gaming | Stable FPS, clean overlays, sensible defaults |
| `competitive` | Ranked sessions | Input latency, process focus, minimal distractions |
| `streaming` | Gameplay capture | Encoder headroom, overlay checks, audio routing notes |

Profiles live in `profiles.json`. Rules live in `optimizer_rules.json`. You can rename, remove, or add rules without changing the runner.

## What It Does

ZeroInput produces a clear launch plan:

1. Reads the selected profile.
2. Expands profile rules into human-readable actions.
3. Checks simple local environment details.
4. Writes an optional session report.
5. Leaves system-level changes to the user.

The default behavior is intentionally conservative. It is a profile and checklist toolkit, not a black-box cleaner.
