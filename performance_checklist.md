# ZeroInput Performance Checklist

Use this checklist before a session when you want repeatable results instead of random tuning.

## Before Launch

- Select one profile: `balanced`, `competitive`, or `streaming`.
- Close downloads, installers, and heavy browser tabs manually.
- Check that the game launcher is already updated.
- Confirm the keyboard, mouse, and headset are on the expected connection path.
- Decide whether capture, replay, or overlays are needed.

## During Warmup

- Watch for uneven frametimes, not only average FPS.
- Keep the game focused while shaders, maps, or assets finish loading.
- Avoid changing multiple settings at once.
- If an issue appears, write one note in the session report.

## After Session

- Compare the current report with the previous run.
- Keep useful rules and remove noisy rules.
- Tune one variable at a time.
- Prefer stable settings over maximum-looking settings.

## Good Defaults

| Goal | Default Habit |
| --- | --- |
| Lower input lag | Keep capture optional and reduce overlay count |
| Stable FPS | Avoid background downloads during gameplay |
| Cleaner testing | Use the same map, scene, or benchmark route |
| Better notes | Save one report per profile change |
